🐍 Snake Game

A browser-based Snake game built from scratch using vanilla HTML, CSS, and JavaScript — no frameworks, no libraries.

Control the snake, eat the food to grow longer, and try to beat your high score — all while the timer counts how long you can survive!

## Features

- 🎮 Classic snake gameplay

- 🍎 Snake grows longer with each food eaten

- 🏆 High score tracking across sessions

- 📊 Live score display

- ⏱️ Timer showing how long you've been playing

## Tech Stack

- HTML5 (Canvas / DOM)

- CSS3

- Vanilla JavaScript

## How to Play

1. Clone or download the repo

2. Open `index.html` in your browser

3. Use arrow keys to control the snake — don't hit the walls or yourself!
